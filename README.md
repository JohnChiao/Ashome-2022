# Ashome V2022

## Introduction - 介绍

Ashome是一个Python轻量级框架，欢迎添加APP！

## Requirements - 必要条件

Ashome内置Python，Flask和Qt环境


Name | Datail
:----------- | :-----------:
.vs|Visual Studio依赖
Application|应用
Lib|模块 
update|升级包
DLLs|.dll扩展
Doc|内置PyManual
include|C库的头文件
Scripts|Python命令行工具

## Installation - 安装

1. 下载Setup.exe
2. 选择路径（或默认）
3. 安装
