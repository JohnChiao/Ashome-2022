## Introduction - 介绍

Ashome是我开发的Python开发工具包，永久开源，可二次开发，内含wxPython，Flask库等。

## Requirements - 必要条件

1. Windows 8.1以上
2. git

## Installation - 安装

1. 下载Zip包
2. 复制

## Development - 开发

### How to update plugins - 如何发布扩展：

1. Fork 本仓库
2. 新增分支，名称规范为：Update_要发布的App名
3. 向分支提交代码
4. 新增Pull Request
